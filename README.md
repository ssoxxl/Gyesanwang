# 계산왕.com 새 사이트

`calc.ssoxxl.com`과 분리된 계산왕.com 전용 정적 사이트다.

## 포지션

- 암산/계산력 훈련 게임
- 생활 속 계산 도구
- 학습형 설명 콘텐츠

## 배포 전 체크

1. 도메인 `계산왕.com` 또는 연결할 실제 호스트명으로 canonical/OG URL 치환
2. 문의 메일 주소 확인
3. Search Console 등록
4. 광고 스크립트는 애드센스 승인 전까지 넣지 않기

## 구조

- `index.html` 메인 홈
- `games/` 계산 게임 섹션
- `calculators/` 계산 도구 섹션
- `learn/` 설명형 학습 콘텐츠 섹션
- 정책성 페이지: `about/`, `contact/`, `privacy/`, `terms/`, `editorial-policy/`
