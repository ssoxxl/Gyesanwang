시간 계산 감각 익히기

시간 계산이 자꾸 헷갈리는 것은 이상한 일이 아닙니다. 우리는 평소 숫자를 10진수처럼 다루는 데 익숙하지만, 시간은 60분 체계라서 같은 방식으로 보면 쉽게 꼬입니다.

분을 먼저 더하고 다시 바꾸기

1시간 20분 + 0시간 45분을 바로 더하면 1시간 65분이 됩니다. 여기서 다시 65분을 1시간 5분으로 바꾸는 과정이 필요합니다. 이 전환이 익숙하지 않으면 계산은 맞아도 표기에서 자주 틀립니다. 그래서 시간 계산은 ‘더한 뒤 다시 정리한다’는 생각이 중요합니다.

총분으로 바꾸면 덜 헷갈립니다

조금만 복잡해지면 모든 시간을 분으로 바꿔버리는 것이 가장 안전합니다. 1시간 20분은 80분, 45분은 45분이니까 둘을 더하면 125분입니다. 여기서 다시 60으로 나누면 2시간 5분이 됩니다. 익숙해질 때까지는 이 방식이 가장 실수가 적습니다.

실생활에서 자주 쓰는 장면

공부 시간 합산, 운동 기록 정리, 업무 시간 계산처럼 시간 더하기는 생각보다 많이 나옵니다. 그래서 시간 계산 감각이 있으면 숫자 감각 전체도 같이 좋아지는 편입니다. 단위가 다른 숫자를 다루는 훈련이 되기 때문입니다.

실전 팁

처음에는 시간 계산기를 같이 써도 괜찮습니다. 계산기 결과를 보면서 직접 분으로 바꾸는 연습을 몇 번 해보면 금방 익숙해집니다. 중요한 것은 ‘60분이 넘어가면 다시 시간으로 넘긴다’는 감각을 몸에 붙이는 일입니다.

시간 계산은 계산력이 부족해서 틀리는 것이 아니라, 단위 전환을 놓쳐서 틀리는 경우가 많습니다.
